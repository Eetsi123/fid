Quickstart
==========
1. Modify `config.py`.
2. `pip install discord.py[voice] youtube_dl`
3. Ensure `ffmpeg` is in `PATH` or next to `fid.py`
3. `python fid.py`

`youtube_dl` is not required if sound is played from a local file.
